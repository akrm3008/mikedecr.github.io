Templates for \(\mathrm{\LaTeX}\) and R Markdown

Document styles, not too complicated, with a personal touch. Now available on Github.

  • \(\mathrm{\LaTeX}\) article (pdflatex). View example
  • R Markdown article (pdflatex). View example
  • TBD: memoir-class documents (e.g. for dissertations).
  • TBD: \(\mathrm{\LaTeX}\) and R Markdown documents built with xelatex.

Templates use Minion and Myriad typeface packages, which won’t be set up for pdflatex unless you have done the work for that. If you want the quick route to enabling those typefaces, build your document with xelatex instead of pdflatex (but N.B. xelatex’s typography isn’t as strong). If you don’t want either, change the typeface package.

Workshops: Math Camp and \(\mathrm{\LaTeX}\)

Materials for these workshops are all open source and available on Github:

  • Math camp for political science graduate students. Introduces important concepts in algebra/pre-calculus, basic linear algebra, set notation and theory, derivative and integral calculus, and probability.
  • \(\mathrm{\LaTeX}\) workshop. What is \(\mathrm{\LaTeX}\), example documents, bibliographies, beamer, workflow integration with statistical software, R Markdown, and Xaringan

Sublime Text User Settings

The Sublime Text editor is a great mix of extensibility and usability for social science, but its text-based settings can be daunting for the uninitialized and a rabbit hole for the detail-obsessed.

My settings can be viewed and pilfered at will from Github. Highlights include:

  • Keybindings (primarily for R and \(\mathrm{\LaTeX}\)) and associated macro files.
  • Code snippets for various languages—mainly R and \(\mathrm{\LaTeX}\), but some for Stan, Stata, HTML, and Markdown.
  • Modified solarized color schemes, available in light, dark, and lastly “gray,” which is a weird hybrid of a few color schemes.
